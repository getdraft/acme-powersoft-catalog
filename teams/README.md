# Teams

One subdirectory per Acme PowerSoft engineering team. Each contains:

- `ONBOARDING.md` — the Admin agent's onboarding briefing for that team's lead
- `sdp.yaml` — the team's committed SoftwareDeploymentPattern, derived from
  their product repo (see Phase 5 of the exercise)

See `EXERCISE_REPORT.md` at the repo root for the disposition assigned to
each team and how it shaped the SDP construction process.
